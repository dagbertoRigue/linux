Outros comandos :

ls -a : exibe arquivos e diretórios ocultos.
ls -f : exibe os diretórios com / no fim
nl : exibe um arquivo com seus números de linhas
wc -l : conta o número de linhas de um arquivo incluindo as em branco e exibe no terminal
wc -w : conta o número de palavras de um arquivo e exibe no terminal
wc -c : exibe o número de bytes de um arquivo
sort : organiza um arquivo em ordem algabética ou numérica
time : exime, em conjunto com um comando, o tempo de processamento de um comando para o sistema, para o usuário e real
uptime : exibe informações de carregamento do sistema
cmp : compara dois arquivos
seq : imprime uma sequência de número com inicío e fim
route -n : exibe a tabela de roteamento IP do Kernel
alias : permite trocar (setar) um nome para um comando
last reboot : exibe informações sobre reinicialização do sistema
history -c : limpa o histórico de comandos
init0 : desliga o sistema
telinit 0 : desliga o sistema
halt : desliga o sistema
whereis : exibe o caminho do programa e seu manual
which exibe o caminho de um programa
